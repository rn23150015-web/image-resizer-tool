EC向け商品画像自動リサイズアプリケーション

ECサイト運営者における「商品画像登録の手間削減」および「サイト表示速度の最適化」を目的としたWebアプリケーションです。

1. 開発背景と解決する課題
ECサイト運営において、商品画像の登録作業は必須ですが、「サイトに合わせた適切なリサイズ」や「画質を保ったままの容量最適化」を手作業で行うことは、運営者の大きな業務負担となっています。
また、不適切なサイズの画像はサイトの表示速度を低下させ、ユーザーの離脱原因になります。
本アプリは、画像をアップロードするだけでECサイトに最適なサイズへ自動処理し、バックオフィス業務の効率化とサイトパフォーマンスの向上を同時に実現するために開発しました。

2. 機能一覧
画像アップロード機能：ファイル選択による画像登録
自動リサイズ・最適化処理：登録された画像を、ECサイトの仕様に合わせた規定サイズへ自動変換

## 3. 使用技術・開発環境
言語：Java 
フレームワーク：Spring Boot
アーキテクチャ：MVC (Model-View-Controller)
開発ツール：Eclipse

4. アーキテクチャとこだわり
本アプリは、当初学習していた「Servlet/JSP/JavaBeans/JDBC」の構成から、実務での標準的な開発手法や保守性の高さを意識し、現在は「Spring Boot」への移行を推進して開発しています。
データの流れを明確にするためMVCモデルを意識した設計を行っております。

5. 今後の展望（Python AI連携計画）
現在はJavaによるWeb機能の構築と自動リサイズ処理を中心に実装していますが、将来的には「Pythonを用いたAI画質向上機能の連携」を計画しています。
アップロードされた低解像度の商品画像を、PythonのAI（超解像技術など）を用いて高画質化してからリサイズ処理に回すことで、ECサイトにおける商品の魅力を最大化し、さらなる売上向上へ貢献するシステムへと進化させる予定です。
